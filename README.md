# MoonMoodTracker

Diese Webanwendung zeigt dem Nutzer,
- die tägliche Mondphase an
- tägliche Astrologie Zitate aus einer Api
- die Möglichkeit seinen Tag zu Journalen

Die Webanwendung ist noch in der Entwicklung.
